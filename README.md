# perfree-base
一个新架子,前后端支持插件化动态插拔的后台管理模板, 文档正在加急编写中...

## 功能简介
* 设计简洁，界面美观
* 前后端支持插件动态化开发,生产环境动态插拔
* 支持系统/插件代码生成
* 支持加载动态权限菜单，按钮级别权限控制
* ......
